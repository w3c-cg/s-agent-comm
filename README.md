# Semantic Agent Communication Community Group

This repository serves as the governance and coordination entry point for the
Semantic Agent Communication Community Group (CG) under W3C.

It hosts the adopted Charter and related coordination materials of the Community
Group.

---

## Charter

The Charter of the Semantic Agent Communication Community Group has been adopted
by Community Group consensus and serves as the governing document of the Group.

The Charter is published at:

https://w3c-cg.github.io/s-agent-comm/

---

## Technical Work and Specifications

Active technical work, specifications, ontologies, and supporting artifacts of
the Community Group are developed and maintained under the following GitHub
organization:

https://github.com/s-agent-comm/

This includes (but is not limited to):

- Semantic Agent Communication Ontology
- AgentIDL specifications
- Conformance and validation artifacts
- Supporting design notes and examples

---

## Repository Structure and Roles

This repository (`w3c-cg/s-agent-comm`) focuses on:

- Community Group governance
- Charter maintenance
- Coordination and process documentation

Technical repositories under `https://github.com/s-agent-comm/` focus on:

- Specification development
- Ontology and schema design
- Validation artifacts and examples

This separation ensures clear governance while allowing technical work to evolve
independently in dedicated repositories.

---

## Participation

Feedback, issues, and contributions are welcome via the relevant repositories
under the `s-agent-comm` organization, in accordance with the W3C Community and
Business Group Process and contribution guidelines.
